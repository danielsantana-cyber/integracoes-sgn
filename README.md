# Central de Suporte SGN

Base de apoio operacional para o suporte, com foco em tratativas, triagem, escalonamento e acompanhamento de problemas recorrentes.

## Conteúdo principal

- Home orientada à decisão rápida
- Fluxo de atendimento
- Direcionamento por equipe
- Erros de integração
- Erros do Benner
- Casos operacionais recorrentes
- Chamados abertos e acompanhamento

## Objetivo

Ajudar o suporte a agir com mais segurança e dar visão útil para gestão.
